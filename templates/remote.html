What is this? -
This site crawls Wikipedia page edit histories and returns a visualization of the frequency of edits made to a particular page. The frequency of edits made to a page are color-coded. Red signifies a relatively high number of edits in a month, green signifies a low number of edits, gray signifies either zero edits, a month that has not yet occurred or a month prior to the page's creation.

What's with the favicon? The Wikipedia favicon rotated 90 degrees looks like a sigma. Sigma is the summation operator. This code summarizes wikipedia edit data.

More info +
More info -
Because the code depends on http requests, which time out in 30 seconds, it is currently only capable of grabbing the most recent few hundred edits. If you query a page with a higher number of edits, your results will display only the most recent activity.

Bugs: This code is extremely slow. If you know how to implement background workers in Flask, or see any other glaring flaws in the source, get in touch.
